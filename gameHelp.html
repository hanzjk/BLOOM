TOWER GAME

    Build up the tower by stacking the building blocks which are dropped from a hanging hook.
    Click/Tap to release the block from the hook.
    Try to drop the blocks right on to the top of the block below.
    Swinging speed of the rope increase when the tower height increases.
    Miss-drops will cost in lives and total lives given are three.


CLUMSY BIRD GAME

    Fly the clumsy bird through the pipes.
    Click/Tap on the screen to make the clumsy bird flap its wings.
    Control the flying height with the speed you tap/click.
    Don't let the bird hit on a pipe it will end the game.
    Try to fly as far as possible.


BUBBLE SHOOTER GAME

    Break down the clusters of same color bubbles by hitting them with the bubble below.
    Set the direction of the ball by handling the stick.
    Clear all the bubbles in the grid to win the game.
    Having more balls in the cluster will add more points to the score.
    Game will be over when the grid is completely filled with the bubbles.


DIRT BIKE GAME

    Drive the bike along the road which comprises of deadly bumps and slopes.
    Press the Up Arrow key to accelerate forward.
    Press the Down Arrow key to accelerate backwards.
    Press the Left and Right Arrow keys to rotate.
    Try to drive as far as possible keeping the balance of the bike.


BREAKOUT GAME

    Break down the tiles by hitting from the ball.
    Avoid the ball been dropped off the pad.
    Move the pad by left and right arrow keys.
    A life will be lost for every dropping off from the pad.
    Game has three levels and break all the tiles to proceed to the next level.
